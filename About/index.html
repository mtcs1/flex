About "the" Design

Trikea is a simple, 3 column layout. It was designed to be lightweight, tableless and conform (and validate) to w3c standards both in css and xhtml.

This design contains 9 static html pages *(see menu on the right) and includes css styles for a wide range of content, menus, forms and elements.

Trikea displays best at any resolution 1024 and over. I did design a pared down 2- column version that works better on smaller monitors or lower resolution and will add or update this design when time allows.

I've released trikea to the public under creative commons copyright so feel free to use, modify and share if you wish.
